## General high-severity

- **Microsoft Defender "RoguePlanet" zero-day (CVE-2026-50656) — Microsoft acknowledges, patch in progress (2026-06-18) [HIGH, patch pending].** The "RoguePlanet" PoC dropped 2026-06-10 by independent researcher "Nightmare Eclipse" amid an ongoing dispute with Microsoft over disclosure / bug-bounty practices. Today Microsoft publicly acknowledged the bug and confirmed they are "actively developing a security patch" but offered no ETA. Technical detail: a race condition in Microsoft Defender's quarantine pipeline (improper link resolution before file access) lets a local low-priv attacker win the race and spawn a SYSTEM-level shell on fully patched Windows 10 and Windows 11. The researcher reports 100% success on some test hardware and variable success on others — typical of TOCTOU race-condition exploits. CVSS reported as 7.8 by Help Net Security (local privilege escalation). Practitioner relevance: this is a local-privesc, not a remote takeover, but it lifts any foothold (phish, malicious download, second-stage from a wider compromise) directly to SYSTEM with no AV detection (it abuses Defender itself). For telco SOC fleets: any Windows-based engineering / OSS / BSS workstation reachable by a compromised insider or initial-access foothold is in scope. Mitigation until patch ships: tight local-logon controls, EDR rules on Defender component file races and unusual SYSTEM-shell spawns, and a watch on Nightmare Eclipse's channels for follow-up drops.
