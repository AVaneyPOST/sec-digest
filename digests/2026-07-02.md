## General high-severity
- **Progress Kemp LoadMaster — CVE-2026-8037 — active exploitation** — [The Hacker News](https://thehackernews.com/2026/07/latest-progress-kemp-loadmaster-pre.html) / [Help Net Security](https://www.helpnetsecurity.com/2026/06/30/simplehelp-vulnerability-exploited-cve-2026-48558/) — 2026-07-01 — **CRITICAL**. OS command injection in `escape_quotes()` (null-termination flaw); unauth root RCE when API is enabled. Affects Kemp LoadMaster GA ≤7.2.63.1 and LTSF ≤7.2.54.17. eSentire's first-round observed attempts failed; PoC public — expect a rapid uptick.
- **Adobe ColdFusion & Campaign Classic — 7× CVSS 10.0** — [BleepingComputer](https://www.bleepingcomputer.com/news/security/adobe-patches-seven-max-severity-coldfusion-campaign-flaws/) / [The Hacker News](https://thehackernews.com/2026/07/adobe-patches-7-cvss-100-flaws-in.html) — 2026-07-01 — **CRITICAL**. Campaign Classic CVE-2026-48286 (incorrect authz → RCE). ColdFusion 2023/2025: CVE-2026-48276/48277/48281/48282/48283/48316 all rated 10.0. Low complexity, no user interaction, no auth. Not yet exploited per Adobe, but 72-hour deployment guidance.
- **Azure CLI password spray — 78 orgs compromised** — [The Hacker News](https://thehackernews.com/2026/07/azure-cli-password-spray-hits-at-least.html) / [SecurityWeek](https://www.securityweek.com/massive-password-spray-campaign-targeting-azure-cli/) — 2026-07-01 (Huntress report) — **HIGH**. Attackers used OAuth ROPC to validate creds against Azure CLI's public app ID over 81M attempts (June 12–26). Conditional Access not covering ROPC = MFA bypass. Hunt for ROPC token issuance in your tenant; disable ROPC where possible.
- **DHS HSIN platform breach confirmed** — [BleepingComputer](https://www.bleepingcomputer.com/news/security/dhs-confirms-hackers-breached-hsin-info-sharing-platform/) / [Nextgov/FCW](https://www.nextgov.com/cybersecurity/2026/06/hackers-breached-dhs-information-sharing-network-people-familiar-say/414534/) — disclosed 2026-07-01 — **HIGH**. Homeland Security Information Network + a linked SharePoint touched by unknown actor late May/early June. Classified systems reportedly untouched; scope of document exfil unclear. Timing overlaps with World Cup security planning.
- **CISA BOD 26-04 deadline: SimpleHelp CVE-2026-48558** — [CISA alert](https://www.cisa.gov/news-events/alerts/2026/06/29/cisa-adds-one-known-exploited-vulnerability-catalog) / [SocRadar](https://socradar.io/blog/cve-2026-48558-simplehelp-oidc-infostealer/) — remediation deadline 2026-07-02 — **CRITICAL**. OIDC identity-token signature not verified; unauth attacker forges token, bypasses MFA. Exploited to deploy TaskWeaver and Djinn Stealer. Patch or take offline.
