## General high-severity

- **IronWorm — Rust npm supply-chain worm with eBPF rootkit, Tor C2, and AI-key sweeper. JFrog disclosure 2026-06-04.** Distributed across ~36 packages published from the compromised `asteroiddao` maintainer account; ELF binary (~976 KB) fires via the `preinstall` hook, packs a custom-modified UPX stub to break signature unpackers, encrypts every internal string with per-call-site keys, and carries an embedded eBPF kernel rootkit to hide its own processes, sockets, and tripwires. Credential harvest: 86 environment variables and 20 file paths covering AWS, GCP, Azure, Vault, Kubernetes, npm, Docker, GitHub, **and every major 2026 AI provider key (Anthropic, OpenAI, Gemini, Cohere, Mistral, Groq, Perplexity, xAI)**; targets Exodus cryptocurrency wallets. Self-propagation: re-publishes trojanised versions of any package the victim's npm token (including Trusted-Publishing OIDC tokens) can publish to. C2 routed through Tor; classed as the Rust-language successor to the Shai-Hulud / Miasma lineage. SlowMist notes Web3 ecosystem as the primary financial target. **Action:** audit npm install logs back to mid-week, rotate every secret in the 86-var / 20-file list, rotate AI-provider keys, re-issue npm Trusted Publishing tokens, scan for eBPF programs and Tor connections on dev/CI hosts. [BleepingComputer](https://www.bleepingcomputer.com/news/security/new-ironworm-malware-hits-36-packages-in-npm-supply-chain-attack/) · [JFrog research](https://research.jfrog.com/post/iron-worm-shai-hulud-rustier-cousin/) · [Dark Reading](https://www.darkreading.com/cyberattacks-data-breaches/rust-written-ironworm-npm-supply-chain) · [Ox Security](https://www.ox.security/blog/ironworm-supply-chain-malware-hits-npm/) · [GBHackers](https://gbhackers.com/ironworm-npm-attack/) · [Phoenix Security technical](https://phoenix.security/ironworm-npm-supply-chain-worm-rust-ebpf-rootkit-tor/).
