## General high-severity

- **RoguePlanet — Windows Defender race-condition LPE, dropped 2026-06-10 by "Nightmare Eclipse" (a.k.a. Chaotic/Dead Eclipse).** Unprivileged local user wins a TOCTOU race in Defender's quarantine/file-handling pipeline; Defender runs as SYSTEM, the redirected operation executes attacker-controlled code at SYSTEM. PoC published on GitHub. Confirmed to work on fully patched Windows 10 and Windows 11 (including Canary Insider Preview) with the June 2026 Patch Tuesday cumulative installed. ThreatLocker independently reproduced against KB5094126. Server-class Windows is believed to be vulnerable to the same primitive with a redesigned attack vector but the dropped PoC does not work as-is on Server. Microsoft's response so far is Defender signature update 1.453.20.0, which adds detection and quarantine for the published PoC — researchers have noted the signature is rudimentary and trivially bypassed with minor code mutation. No root-cause patch and no MSRC-assigned CVE were available at time of publication; some third-party trackers have linked the issue to CVE-2026-47281 (Visual Studio Code EoP, MSRC) but the direct mapping is contested in the primary coverage and should be treated as unverified until MSRC publishes a Defender-specific advisory. Mitigation today: application allowlisting (AppLocker / WDAC) blocks the dropped PoC; raise alert priority on any local SYSTEM token elevation from non-system processes. [CRITICAL on standard end-user fleets without app control. Active exploitation in the wild not yet reported as of 2026-06-11.]

- **Ivanti Sentry CVE-2026-10520 (CVSS 10.0) + CVE-2026-10523 (CVSS 9.9) — watchTowr drops technical write-up + working PoC on 2026-06-10.** Sentry is the secure mobile gateway sitting between Ivanti EPMM/MobileIron Core and corporate Exchange/PKI/AD; compromise = full visibility of every managed-device session and a credential-laundering position. CVE-2026-10520 is an unauthenticated OS command injection in the ConfigServiceController class, reachable via POST to `/mics/api/v2/sentry/mics-config/handleMessage` — payload is interpreted as a MICS configuration command by a backend component and executed as root. CVE-2026-10523 is a separate pre-auth authentication bypass that lets a remote attacker create an arbitrary admin account. Patched in Sentry R10.5.2, R10.6.2, R10.7.1 (released 2026-06-09). Ivanti's advisory said no known in-the-wild exploitation at disclosure — that assumption now has hours to live, not days. watchTowr noted the bug class and reachability are essentially a re-tread of the 2023 Sentry pre-auth RCE (CVE-2023-38035) and that patched-and-forgotten Sentries are likely still hot. Anyone running pre-fix Sentry on the public internet should treat this as compromised until proven otherwise; pull internet exposure, rotate certificates and admin credentials, and audit `/mics` endpoint access logs from 2026-06-09 onward. [CRITICAL]
