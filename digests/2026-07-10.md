## TL;DR
- **Microsoft ships out-of-band fix for RoguePlanet Defender zero-day CVE-2026-50656** — engine 1.1.26060.3008 lands 29 days after the "Nightmare Eclipse" PoC exposed a race-condition SYSTEM shell in Microsoft Defender's Malware Protection Engine. CVSS 7.8 local EoP, but the PoC works with real-time protection on or off — the story now dominates the 09 Jul news cycle. Patch push is silent-auto for most endpoints, but confirm the engine version — a stale offline VDI image will still be vulnerable.
- **China-nexus UAT-7810 expands LapDogs ORB with LONGLEASH / DOGLEASH / JARLEASH families (Cisco Talos, 08 Jul, coverage into 09 Jul)** — new C and Java backdoors on top of SHORTLEASH; infrastructure overlaps with early-2026 ASUS AiCloud router exploitation (CVE-2025-2492) and services downstream China-nexus APTs including UAT-5918. Telecom-adjacent: this is edge/CPE router compromise being converted into rentable relay infrastructure for other actors' operations.
- **GhostApproval symlink flaw hits six AI coding agents (Wiz, 08 Jul, coverage into 09 Jul)** — Amazon Q Developer (CVE-2026-12958), Cursor (CVE-2026-50549), Claude Code, Augment, Google Antigravity, Windsurf. Malicious repo's `project_settings.json` symlink resolves to `~/.ssh/authorized_keys`; the approval dialog names the friendly file, not the symlink target. Three vendors patched, two haven't, Anthropic disputes it's a bug. If you allow developers to run agents on untrusted repos, treat this as a supply-chain vector.
- **CISA KEV BOD 26-04 deadline expires today (10 Jul)** for the 07 Jul batch: ColdFusion CVE-2026-48282 (CVSS 10.0), Langflow CVE-2026-55255 (CVSS 9.9), JoomShaper SP Page Builder CVE-2026-48908, Joomlack Page Builder CVE-2026-56290. FCEB agencies out of runway; commercial ops running any of these on the internet should treat the deadline as their own.
- **Telecom SS7/Diameter/GTP/5G-SBA/N32: no fresh disclosures in-window.** UAT-7810 router-CPE work is the only telecom-adjacent fresh item; Ubiquiti + KDDI + Salt Typhoon remain rolling background.

## General high-severity
- **Microsoft patches RoguePlanet Defender zero-day CVE-2026-50656 (CVSS 7.8, local EoP)** — [HIGH] Delivered via Malware Protection Engine 1.1.26060.3008 on 08 Jul; the 09 Jul news cycle picked up the "29 days from public PoC to fix" framing. Vulnerability is a race condition in the Defender scanning engine that lets a local unprivileged process spawn a SYSTEM shell; PoC works whether real-time protection is on or off. Fully-patched Windows 10 / 11 was affected. Microsoft rates it "Exploitation More Likely" and made the fix flow silent-auto via engine update, but any device that stopped receiving Defender updates (offline VDI, isolated OT bastion, air-gapped engineering laptop, forensic image) is still vulnerable — verify the engine version. Sources: [SecurityWeek](https://www.securityweek.com/microsoft-patches-defender-rogueplanet-vulnerability/), [Security Boulevard](https://securityboulevard.com/2026/07/microsoft-fixes-rogueplanet-zero-day-in-defender/), [Security Affairs](https://securityaffairs.com/195016/security/microsoft-fixed-defender-flaw-rogueplanet-cve-2026-50656.html), [Cybersecurity News](https://cybersecuritynews.com/rogueplanet-defender-zero-day-vulnerability/), [Kudelski Security](https://kudelskisecurity.com/research/rogueplanet-zero-day-ms-defender-privilege-escalation), [Morphisec](https://www.morphisec.com/blog/microsoft-defender-zero-day-rogueplanet-when-your-detector-becomes-the-attack-surface/), [SC Media](https://www.scworld.com/brief/microsoft-releases-patch-for-defender-zero-day-vulnerability-rogueplanet).
- **GhostApproval — six AI coding agents let malicious repos redirect writes via symlink (Wiz, 08 Jul, coverage into 09 Jul)** — [HIGH] Repo ships a symlink named `project_settings.json` that points to `~/.ssh/authorized_keys` (or `~/.zshrc`, or an npm postinstall shim). The agent's "may I write to project_settings.json?" prompt is truthful about the *name* but hides the *target* — even when the agent internally reasons that the file is really something else (Wiz demonstrated this with Claude Code). One click and the write hits your SSH keys or shell RC file. Affected: Amazon Q Developer (CVE-2026-12958, fixed in language-server 1.69.0 on 27 May 26), Cursor (CVE-2026-50549, fixed in v3.0 on 05 Jun 26), Claude Code (Anthropic disputes it's a bug), Augment, Google Antigravity, Windsurf (patch status per vendor). Practitioner impact: developers running agents against untrusted repos is now an untriaged supply-chain vector; audit fleet versions and disable auto-approve on file writes. Sources: [The Hacker News](https://thehackernews.com/2026/07/ghostapproval-symlink-flaws-could-let.html), [SecurityWeek](https://www.securityweek.com/ai-coding-tools-tricked-into-hacking-developer-machine-via-decades-old-technique/), [The Register](https://www.theregister.com/security/2026/07/08/bug-in-top-ai-coding-agents-shows-that-unix-era-security-headaches-never-really-die/5268025), [Cybersecurity News](https://cybersecuritynews.com/ghostapproval-vulnerability/amp/), [GBHackers](https://gbhackers.com/ghostapproval-attack-impacts-amazon-q-claude-code-cursor-google-antigravity-and-windsurf/), [Infosecurity Magazine](https://www.infosecurity-magazine.com/news/ghostapproval-flaw-ai-coding/).
- **CISA KEV — 07 Jul batch deadline expires today, 10 Jul** — [CRITICAL] Rolling: Adobe ColdFusion CVE-2026-48282 (CVSS 10.0, path-traversal → RCE, active exploitation from 103.207.14.220), Langflow CVE-2026-55255 (CVSS 9.9, authorization bypass through user-controlled key), JoomShaper SP Page Builder CVE-2026-48908 (unrestricted file upload), Joomlack Page Builder CVE-2026-56290 (improper access control). BOD 26-04 mandates FCEB fixes today; Langflow is the first AI-agent platform on KEV. If you can't patch ColdFusion in-window, disable RDS. Sources: [CISA alert](https://www.cisa.gov/news-events/alerts/2026/07/07/cisa-adds-three-known-exploited-vulnerabilities-catalog), [The Hacker News](https://thehackernews.com/2026/07/cisa-adds-4-actively-exploited-adobe.html), [TechTimes](https://www.techtimes.com/articles/319918/20260708/cisa-adds-first-ai-agent-platform-kev-sets-thursday-deadline-4-cves.htm).
- **Mount Royal University — CMD Organization ransomware, delete-after-steal, $1.9M demand (disclosed 09 Jul)** — [NOTABLE] Threat actor exfiltrated then deleted 10 TB of student and administrative data — deleting the victim copy means backups aren't a get-out-of-jail card, because the attacker's copy is the only remaining one. Ask is ~4× the group's average ($580K), suggesting a bespoke pricing on the target. CMD Organization infra first appeared late March 2026; recovery reportedly weeks-to-months. Sources: [TechTimes](https://www.techtimes.com/articles/320031/20260709/hackers-stole-then-deleted-mount-royal-university-data-students-get-no-credit-monitoring.htm).
