## TL;DR
- **Progress Software tells ShareFile customers to power off on-prem Storage Zone Controllers now** — 10 Jul emergency notice cites a "credible external security threat"; no CVE assigned yet, no confirmation whether this is a fresh zero-day or continued exploitation of the April watchTowr chain (CVE-2026-2699 / -2701). - **CISA KEV adds two Joomla-extension file-upload RCEs, both CVSS 10.0** — iCagenda ≤ 4.0.7 (CVE-2026-48939) and Balbooa Forms ≤ 2.4.0 (CVE-2026-56291). Both patched last month but pre-auth PHP upload → webshell in one HTTP POST, PoC public for iCagenda. Standard KEV BOD 26-04 clock started.

Add PoisonX driver hash to your BYOVD blocklist today; Microsoft's HCK signing pipeline is now a public trust-abuse surface again.
- **Injective Labs SDK — npm supply-chain compromise, mnemonic + private-key exfil** — @injectivelabs/sdk-ts v1.20.21 (published 8 Jul via compromised GitHub) siphons wallet seed phrases through a fake telemetry HTTP POST; 17 sibling Injective packages pin the poisoned version. ~50k weekly downloads. Detected + disclosed 10 Jul. If you have any Injective SDK version in a lockfile since 8 Jul, treat any devbox key material as burned.
- **Telecom SS7/Diameter/GTP/5G-SBA/N32/SEPP/IMS/XCAP: no fresh disclosures in-window.** The only telecom-adjacent development is the Dutch Politie's update on the Odido carrier breach — investigators now say a Dutch national is likely involved and released the audio of the customer-service vishing call used to seed the intrusion.

## General high-severity
- **Progress ShareFile — emergency shutdown notice, "credible external security threat"** — [CRITICAL] Progress emailed Storage Zone Controller customers on 10 Jul telling them to power off on-prem servers immediately. No CVE yet, no public technical detail, no confirmation on whether this is a fresh zero-day or continued exploitation of the April watchTowr chain (CVE-2026-2699 / CVE-2026-2701 — ~30k exposed instances at that time). Story surfaced via a Reddit r/sysadmin post, which BleepingComputer and The Hacker News picked up in-window. Treat as an active-incident advisory: shut down internet-exposed Storage Zone Controllers, snapshot for IR, wait for the technical bulletin. Sources: [BleepingComputer](https://www.bleepingcomputer.com/news/security/progress-urges-sharefile-customers-to-shut-down-servers-over-credible-threat/), [The Hacker News](https://thehackernews.com/2026/07/urgent-progress-tells-sharefile.html), [SC Media](https://www.scworld.com/brief/progress-software-warns-sharefile-users-of-external-security-threat).
- **CISA KEV — two Joomla extension file-upload → RCEs, both CVSS 10.0 (added 10 Jul)** — [CRITICAL] iCagenda ≤ 4.0.7 (CVE-2026-48939, unrestricted PHP upload via event-submission form → unauth RCE; patched in 4.0.8 released 15 Jun; public GitHub PoC) and Balbooa Forms ≤ 2.4.0 (CVE-2026-56291, unauth arbitrary file upload → RCE; patched in 2.4.1). Both are the classic Joomla-extension pattern: front-facing form, no MIME/extension check, webshell dropped and persistent in one POST. BOD 26-04 clock starts today. Sources: [CISA alert](https://www.cisa.gov/news-events/alerts/2026/07/10/cisa-adds-two-known-exploited-vulnerabilities-catalog), [Balbooa writeup](https://mysites.guru/blog/balbooa-forms-unauthenticated-file-upload-flaw), [CIRCL vuln entry](https://vulnerability.circl.lu/vuln/CVE-2026-48939).
- **"GodDamn" ransomware — Beast/Monster rebrand deploys Microsoft-HCK-signed "PoisonX" BYOVD driver to blind EDR** — [CRITICAL] Symantec Threat Intel published 10 Jul; The Hacker News, Dark Reading, SC Media, Security Affairs all carried within the window. PoisonX was uploaded to GitHub 7 April by user "oxfemale" as a "research tool"; it passed Microsoft's Hardware Compatibility signing review, giving the ransomware crew a validly-signed kernel driver that terminates EDR-agent processes on load. Symantec's telemetry: ~10 hosts inside a single US victim in early June, delivery via AnyDesk + NirSoft credential harvester before the ransomware drop; first public sighting 21 May. Blocklist the driver hash today; treat Microsoft-HCK-signed kernel modules as a live BYOVD surface again. Sources: [Symantec / Broadcom](https://www.security.com/threat-intelligence/goddamn-ransomware-beast-rebrand), [The Hacker News](https://thehackernews.com/2026/07/goddamn-ransomware-uses-poisonx-driver.html), [Dark Reading](https://www.darkreading.com/cyberattacks-data-breaches/goddamn-ransomware-byovd-smite-companies), [Security Affairs](https://securityaffairs.com/195042/).
- **Injective Labs SDK — npm supply-chain compromise, wallet key stealer** — [HIGH] @injectivelabs/sdk-ts v1.20.21 was published 08 Jul via a compromised GitHub credential; on install it exfiltrates mnemonic seed phrases and private keys via a fake telemetry HTTP POST. 17 sibling `@injectivelabs/*` packages transitively pin the poisoned version; ~50k weekly downloads on the affected fan-out. Detected by Socket, Ox Security and StepSecurity; disclosure wave hit 10 Jul. If any Injective SDK version has been pulled into a build or a dev workstation since 8 Jul, treat any key material or wallet on that host as burned. Sources: [BleepingComputer](https://www.bleepingcomputer.com/news/security/injective-sdk-on-npm-infected-with-cryptocurrency-wallet-stealer/), [The Hacker News](https://thehackernews.com/2026/07/injective-labs-github-compromise-pushes.html).
- **"Helix" — new data-extortion crew running the BlackFile playbook against SharePoint** — [NOTABLE] ReliaQuest post ~09–10 Jul, BleepingComputer coverage 10 Jul. Attack chain: vishing → device-code phishing → rogue MFA registration → SharePoint enumeration and exfil. Tradecraft and infrastructure overlap materially with BlackFile (shuttered April) and with ShinyHunters; treat as continuation, not a new capability. Detection focus: rogue MFA method additions and device-code sign-in from unusual UAs. Sources: [ReliaQuest](https://reliaquest.com/blog/threat-spotlight-helix-new-name-in-data-extortion-ecosystem), [BleepingComputer](https://www.bleepingcomputer.com/news/security/new-helix-vishing-group-emerges-in-sharepoint-data-theft-attacks/).
- **JetBrains TeamCity 2026.1.2 — CVE-2026-59793 (CVSS 8.8) + CVE-2026-59794 (CVSS 7.3)** — [HIGH, sub-9.0] Arbitrary file access via Perforce VCS integration and stored XSS via agent-reported data on the cloud-profile page. Below the digest's 9.0 threshold but flagged because a multi-tenant TeamCity is a fat pipeline compromise: file-read from another tenant's Perforce workspace is enough to lift build secrets. Sources: [NVD CVE-2026-59793](https://nvd.nist.gov/vuln/detail/CVE-2026-59793), [NVD CVE-2026-59794](https://nvd.nist.gov/vuln/detail/CVE-2026-59794).
