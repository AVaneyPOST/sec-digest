## General high-severity

- **HTTP/2 Bomb — CVE-2026-49975 — remote unauthenticated DoS in default HTTP/2 configurations.** Researchers at Calif (with OpenAI Codex) chained HPACK header-table amplification with Slowloris-style connection hold; measured amplification 70:1 (nginx) to 5,700:1 (Envoy, ~32 GB exhausted in ~10 s from a single residential link). Affects nginx, Apache httpd, Microsoft IIS, Envoy, and Cloudflare Pingora. Fixes: nginx 1.29.8 (new `max_headers` directive), Apache mod_http2 v2.0.41. **No fix yet for IIS, Envoy, or Cloudflare Pingora at disclosure.** Disclosure 2026-06-03 via oss-security; vendors notified 2026-05-27. [oss-security](https://www.openwall.com/lists/oss-security/2026/06/03/3) · [SecurityWeek](https://www.securityweek.com/http-2-bomb-exploit-knocks-web-servers-offline-in-seconds/) · [The Hacker News](https://thehackernews.com/2026/06/new-http2-bomb-vulnerability-allows.html) · [Calif blog](https://blog.calif.io/p/codex-discovered-a-hidden-http2-bomb). Date: 2026-06-03. Tag: HIGH.
- **Miasma / @redhat-cloud-services npm supply-chain attack — continued amplification 2026-06-03.** Compromise of a Red Hat employee's GitHub account let attackers push orphan commits across at least 32 packages (96 versions) under the `@redhat-cloud-services` org; ~117k weekly downloads cumulatively. Preinstall script runs an obfuscated Shai-Hulud descendant ("Miasma: The Spreading Blight") that exfiltrates GitHub / npm / AWS / GCP / Azure tokens and attempts to self-propagate to other packages the victim can publish. Initial Wiz/Microsoft disclosures landed 2026-06-01/02; Hacker News, Snyk, JFrog, and iTnews republished with new technical detail 2026-06-03. Relevant to any DevOps pipeline pulling from npm — including telco / cloud-native NF build chains. [The Hacker News](https://thehackernews.com/2026/06/miasma-supply-chain-attack-compromises.html) · [Snyk](https://snyk.io/blog/miasma-supply-chain-attack-malicious-code-redhat-cloud-services-npm-packages/) · [JFrog](https://research.jfrog.com/post/shai-hulud-miasma-redhat-cloud-services/) · [Wiz](https://www.wiz.io/blog/miasma-supply-chain-attack-targeting-redhat-npm-packages). Date: 2026-06-03 (continued coverage); 2026-06-01 (initial Wiz). Tag: HIGH.
- **WeedHack MaaS Minecraft infostealer — McAfee Labs disclosure, 116k+ infected.** Active since January 2026; distributed via malicious Minecraft mods/clients/cheats over YouTube + SEO poisoning. 3,820 unique malicious JARs and 240+ distribution URLs identified. Free tier = infostealer; $5/mo (or $24.99 lifetime) adds remote shell, keylogger, webcam capture, and remote file manager. Most victims US / DE / IN / UK. Note: McAfee reports operators are predominantly teenage/young-adult and have weaponised webcam access for harassment campaigns on Telegram. Disclosure 2026-06-03. [Help Net Security](https://www.helpnetsecurity.com/2026/06/03/weedhack-minecraft-malware-campaign/) · [BleepingComputer](https://www.bleepingcomputer.com/news/security/over-116-000-minecraft-systems-infected-in-weedhack-malware-campaign/) · [The Hacker News](https://thehackernews.com/2026/06/weedhack-attacks-minecraft-users.html). Date: 2026-06-03. Tag: NOTABLE.
