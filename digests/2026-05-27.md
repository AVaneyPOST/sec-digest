## General high-severity

- **CVE-2026-45659 — Microsoft SharePoint Server RCE (deserialization)** [HIGH]
  - CVSS 8.8. Affects SharePoint Server Subscription Edition, 2019, and Enterprise Server 2016. Authenticated attacker with Site Member (PR:L) can execute code over the network. MSRC published advisory 2026-05-21; broader coverage 2026-05-26. No public PoC reported in window; Microsoft rates "less likely to be exploited" but on-prem SharePoint is a known APT magnet.
  - Source: https://www.helpnetsecurity.com/2026/05/26/sharepoint-vulnerability-cve-2026-45659/ — date: 2026-05-26
  - Source: https://thehackernews.com/2026/05/microsoft-patches-sharepoint-rce-flaw.html — date: 2026-05-26

- **CVE-2026-9082 — Drupal core SQL injection (SA-CORE-2026-004), active mass exploitation** [CRITICAL]
  - Disclosed 2026-05-19; working exploit within ~1 hour of patch (2026-05-20); CISA added to KEV on 2026-05-22 with FCEB deadline 2026-06-05. Reports of >15,000 attempts hitting ~6,000 sites across 65 countries within 48h. Continues to be exploited in this window.
  - Source: https://www.tenable.com/blog/cve-2026-9082-highly-critical-sql-injection-vulnerability-in-drupal-core-sa-core-2026-004

## Worth watching

- **Salt Typhoon (PRC, MSS-linked) — continuing telecom espionage** — Still the highest-impact ongoing telecom story. FBI February 2026 update confirmed activity remains live, basic config errors + known CVEs are still entry points. Norway and Singapore added to confirmed-victim list earlier this year. No new public victim attribution in this 24h window.
- **Citizen Lab "Bad Connection" (2026-04-23) — SS7/Diameter and SIMjacking surveillance** — Documented >500 location-tracking attempts (Nov 2022–2025) via SS7 fallback to Diameter, plus parallel SIMjacker-style SMS-to-SIM campaign. Vendors abused 019Mobile (IL), a UK provider, and a Channel Islands operator as ingress points. Practitioners should re-validate FS.11/FS.19 control coverage and SIM OTA filtering. Source: https://citizenlab.ca/research/uncovering-global-telecom-exploitation-by-covert-surveillance-actors/
- **CVE-2026-0300 PAN-OS User-ID Authentication Portal RCE** — Pre-auth root RCE, CVSS 9.3, exploited since at least April 2026 (CL-STA-1132 cluster). Patches rolling since 2026-05-13. Still an unmitigated risk on internet-exposed captive portals — review any operator perimeter using PAN-OS.
- **Cisco Catalyst SD-WAN CVE-2026-20182** — Limited exploitation reported around 2026-05-14 (Catalyst SD-WAN 20.9–20.18 and 26.1). Relevant for telco branch/aggregation and managed enterprise services.
- **RCS / iMessage PhaaS smishing (Chinese-language ecosystem)** — Ongoing trend through 2026, accelerating: phishing-as-a-service kits routing lures via RCS and iMessage to dodge SMS-layer carrier filters. Operationally important for fraud and abuse teams; not a new disclosure but the volume is climbing.
- **5G N32/SEPP TLS 1.3 0-RTT FS research** — Academic work proposing 0-RTT forward secrecy for cross-border roaming N32 still in pre-print circulation; relevant for SEPP roadmap discussions but not deployable.
