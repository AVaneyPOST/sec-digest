## General high-severity

- **CISA KEV: CVE-2022-0492 — Linux kernel cgroups v1 `release_agent` improper authentication.** Container-escape / local priv-esc primitive. Old bug (originally fixed 2022), but evidence of active exploitation in 2026. Relevant for any unpatched container hosts including telco edge compute / MEC nodes. [CISA alert](https://www.cisa.gov/news-events/alerts/2026/06/02/cisa-adds-two-known-exploited-vulnerabilities-catalog). Date: 2026-06-02. Tag: HIGH.
- **CISA KEV: CVE-2025-48595 — Android Framework integer overflow → LPE.** Google's 2026-06-01 Android Security Bulletin notes "limited, targeted exploitation." No user interaction; local vector (likely malicious app). Affects Android 14 / 15 / 16 / 16 QPR2. CVSS 8.4. AOSP patches within 48h of bulletin. Significant for any mobile-implant or telco-issued-handset threat model. [Help Net Security](https://www.helpnetsecurity.com/2026/06/02/android-vulnerability-exploited-cve-2025-48595/) · [BleepingComputer](https://www.bleepingcomputer.com/news/security/google-fixes-one-actively-exploited-android-zero-day-124-flaws/) · [CISA alert](https://www.cisa.gov/news-events/alerts/2026/06/02/cisa-adds-two-known-exploited-vulnerabilities-catalog). Date: 2026-06-02 (KEV add); 2026-06-01 (Google bulletin). Tag: HIGH.
- **Windows Netlogon RCE (CVE-2026-41089) — ongoing exploitation reporting.** CVSS 9.8 stack-based buffer overflow, unauth, all supported Windows Server versions including 2025. Patched in 2026-05-13 Patch Tuesday; CCB Belgium escalated and follow-up coverage continued through 2026-06-02. Domain-controller takeover primitive — patch latency here is the entire risk. [SecurityWeek](https://www.securityweek.com/critical-windows-netlogon-vulnerability-in-attackers-crosshairs/) · [BleepingComputer](https://www.bleepingcomputer.com/news/microsoft/critical-windows-netlogon-remote-code-execution-flaw-now-exploited-in-attacks/). Date: 2026-06-02 (most recent in-window coverage). Tag: CRITICAL.
