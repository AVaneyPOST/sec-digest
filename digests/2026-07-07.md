## Telecom

- **VodafoneThree SMS firewall — 2M banking-impersonation SMS blocked since Aug 2025** [NOTABLE] — 2026-07-06
  VodafoneThree (UK) published stats on its bespoke SMS-firewall ruleset built with MEF / Cyber Defence Alliance / Barclays / Co-op / TSB. Bespoke sender-ID + content rules stacked into the SMS firewall to distinguish legit bank brand traffic from smishing; ~25% uplift on baseline blocks. Useful data point for red-teaming A2P routing evasion, sender-ID spoofing, and content-heuristic bypass against UK operator SMS firewalls.
  Sources: [Total Telecom](https://totaltele.com/vodafonethree-blocks-over-2-million-fraud-attempts-for-banking-sector/) · [ISPreview](https://www.ispreview.co.uk/index.php/2026/07/vodafonethree-blocks-over-2-million-uk-banking-sector-fraud-attempts.html) · [Light Reading](https://www.lightreading.com/wireless/eurobites-vodafonethree-blocks-2-million-banking-scam-texts)

## General high-severity

- **CVE-2026-53359 "Januscape" — KVM guest-to-host escape (Linux)** [HIGH] — 2026-07-06
  Use-after-free in x86 KVM shadow MMU code, present ~16 years. First public guest→host trigger reproducible on both Intel and AMD hosts. PoC (Hyunwoo Kim / kvmCTF) panics the host; a separate unreleased exploit reportedly achieves host RCE. Stable-kernel fixes shipped 2026-07-04. Directly relevant to any multi-tenant KVM footprint — cloud, telco NFV/CNF hosts included.
  Sources: [The Hacker News](https://thehackernews.com/2026/07/16-year-old-linux-kvm-flaw-lets-guest.html) · [Security Online](https://securityonline.info/januscape-kvm-escape-cve-2026-53359-poc/)

- **Opera GX — "GX Mods" auto-install + CSS-injection exfil** [NOTABLE] — 2026-07-06
  Malicious sites could silently install an Opera GX mod and CSS-inject to exfiltrate data; PoC extracted Gmail address of any signed-in visitor. Patched in 130.0.5847.89 (Opera P1, $5k bounty). No exploitation in the wild reported.
  Sources: [The Hacker News](https://thehackernews.com/2026/07/opera-gx-flaw-let-malicious-sites-auto.html) · [Opera Security blog](https://blogs.opera.com/security/2026/07/security-fix-gx-mods-vulnerability/)

- **Operation DragonReturn — suspected China-nexus APT vs Indian tax infrastructure** [NOTABLE] — 2026-07-06
  Seqrite Labs: spear-phish impersonating India's Income Tax Department (fake ITR AY2026-27 "offline utility") drops DcRAT via a multi-stage loader with process injection and AMSI bypass. Infrastructure overlaps Silver Fox.
  Sources: [The Hacker News](https://thehackernews.com/2026/07/suspected-china-nexus-hackers-use-fake.html) · [Seqrite Labs](https://www.seqrite.com/blog/operation-dragonreturn-china-nexus-cyber-espionage-campaign-targeting-govt-of-india-mof-tax-infrastructure-via-multi-stage-dcrat-deployment/)

- **QuimaRAT — new cross-platform Java RaaS** [NOTABLE] — 2026-07-06
  Modular Java RAT targeting Windows / Linux / macOS with encrypted plugin loader, JNA native calls, fileless shellcode on Windows, per-OS persistence. Sold at $150/month → $1,200 lifetime. Watch for early campaigns.
  Source: [The Hacker News](https://thehackernews.com/2026/07/new-java-based-quimarat-maas-built-to.html)
