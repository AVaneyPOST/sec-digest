## General high-severity

- **FBI flash: Silent Ransom Group physically infiltrating law firm offices** — Group (aka Luna Moth, Chatty Spider, UNC3753) blends voice phishing posing as IT and, when that fails, dispatches operatives in person to plug USB drives into endpoints. Few host artifacts (legit RMM tools used). Sectoral focus: law firms, but also insurance/finance/healthcare. [BleepingComputer](https://www.bleepingcomputer.com/news/security/fbi-warns-of-silent-ransom-group-in-person-data-theft-attacks/) · [CyberScoop](https://cyberscoop.com/fbi-warning-silent-ransom-group-law-firms/) · [The Register](https://www.theregister.com/security/2026/05/27/fbi-crooks-enter-legal-offices-and-steal-data-via-usb-drive/5247212) · Date: 2026-05-27 · **HIGH**

- **CISA KEV: CVE-2026-48172 — LiteSpeed cPanel plugin priv-esc to root (CVSS 10.0)** — Any cPanel user can call `lsws.redisAble` to run arbitrary scripts as root. Versions 2.3–2.4.4 vulnerable; fixed in 2.4.5 / WHM plugin 5.3.1.0. Federal remediation due 2026-05-29. Mass opportunistic exploitation against shared-hosting tenants. [The Hacker News](https://thehackernews.com/2026/05/litespeed-cpanel-plugin-cve-2026-48172.html) · [SecurityWeek](https://www.securityweek.com/cisa-urges-immediate-patching-of-exploited-litespeed-cpanel-plugin-zero-day/) · Date: 2026-05-27 · **CRITICAL**

- **Ghost CMS CVE-2026-26980 — unauth SQLi, CVSS 9.4, ClickFix campaign** — Affects Ghost 3.24.0–6.19.0. Unauthenticated attacker reads arbitrary DB rows including admin API keys, then injects JS into posts to serve a fake Cloudflare ClickFix lure that drops Windows malware. >700 domains hit per XLab/Qianxin (incl. Harvard, Oxford, Auburn, DuckDuckGo blog). Fix: 6.19.1; rotate admin API keys, audit posts. [BleepingComputer](https://www.bleepingcomputer.com/news/security/ghost-cms-sql-injection-flaw-exploited-in-large-scale-clickfix-campaign/) · [The Hacker News](https://thehackernews.com/2026/05/ghost-cms-cve-2026-26980-exploited-to.html) · Date: 2026-05-27 · **CRITICAL**

- **Laravel-Lang supply-chain compromise (broader reporting)** — Push-access compromise of the `laravel-lang` GitHub org; attacker rewrote every git tag across multiple Composer packages on 2026-05-22 22:32 UTC. Malicious `src/helpers.php` autoloads, drops `DebugElevator` (Windows binary) from `flipboxstudio[.]info` that exfiltrates AWS/GCP/Azure creds, GitHub tokens, Slack/Stripe tokens, SSH keys, `.env`, JWTs, K8s/Vault secrets, crypto seed phrases. Coverage broadened over the May 27 cycle. [BleepingComputer](https://www.bleepingcomputer.com/news/security/laravel-lang-packages-hijacked-to-deploy-credential-stealing-malware/) · [StepSecurity advisory](https://www.stepsecurity.io/blog/laravel-lang-supply-chain-attack) · [The Hacker News](https://thehackernews.com/2026/05/laravel-lang-php-packages-compromised.html) · Initial attack 2026-05-22; broader disclosure 2026-05-27 · **HIGH**

- **KnowledgeDeliver LMS — CVE-2026-5426, hardcoded ASP.NET machine key → unauth RCE via ViewState** — Japanese LMS vendor Digital Knowledge. Same machine key shared across customer deployments. Exploited as a 0-day to plant the Godzilla web shell, then sideload Cobalt Strike beacon and inject a fake "install plugin" lure into the LMS UI. All deployments predating 2026-02-24 are exposed. [SecurityWeek](https://www.securityweek.com/hackers-exploited-knowledgedeliver-zero-day-for-web-shell-deployment/) · [BleepingComputer](https://www.bleepingcomputer.com/news/security/knowledgedeliver-flaw-exploited-as-a-zero-day-to-install-web-shells/) · [The Hacker News](https://thehackernews.com/2026/05/knowledgedeliver-lms-flaw-exploited-to.html) · Date: 2026-05-27 · **HIGH**
